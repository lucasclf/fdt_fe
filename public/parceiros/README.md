Coloque aqui as imagens dos NPCs.

Regra usada pelo projeto:
- o arquivo deve se chamar exatamente igual ao nome do NPC + ".png"
- exemplos:
  - Abner.png
  - Mutuca.png
  - Arlo “Olho de Leite”.png
  - Nada Mais que a Verdade Reston.png
  - Rot'Kn.png

Atenção:
- espaços, acentos, aspas e maiúsculas/minúsculas importam em hospedagem estática.
- se preferir nomes sem acento no futuro, altere o campo "imagem" em src/data/parceiros.ts.
